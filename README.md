Este es el proyecto de simulación en OMNeT++ versión 6.0.3
1. Debe instalarse OMNeT++ 6.0.3 para el correcto funcionamiento, debe tenerse en cuenta instalar en la carpeta de instalación C:/
2. Debe instalarse la versión de INET 4.4.0, instalar en una carpeta FUERA de la carpeta de instalación de OMNeT++ y hacer build en modo release 
3. Debe instalarse la versión de FLoRa 1.1.0, instalar en una carpeta FUERA de la carpeta de instalación de OMNeT++ y hacer build en modo release
4. Debe hacerse build al proyecto actual en modo release
5. El archivo omnet_runner.py contiene la ruta hacia la carpeta donde OMNeT++ está instalado, al igual que INET y FLoRa, 
deberán ser reemplazadas por las respectivas rutas del usuario donde haya instalado el stack
6. La imagen de la mina se debe cambiar directamente en /omnet-6.0.3/images/background/ -la imagen debe llamarse "mina.jpg"-
